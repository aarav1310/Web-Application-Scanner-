#Install the required dependencies:
pip install -r requirements.txt

Run the script:
python web_scanner.py <url>

Replace `<url>` with the URL of the web application you want to scan.

The script will perform a scan using OWASP ZAP and display the results.

## Requirements

- Python 3.x
- `pyzap` library (install using `pip install pyzap`)
